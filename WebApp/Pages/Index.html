Accessible Mixed Effects Models via a Guided Interface

Yaron Ziner

Lior Wolf
Tel-Aviv University Tel-Aviv University
Abstract: We present a novel software framework aimed at guiding users with the creation and analysis of mixed linear regression models. The support occurs throughout the entire process: model creation, model fitting, interpreting the results, and reporting the outcome of the analysis. The system makes several model validity checks and alerts the user regarding the suitability of various modeling options. The software thus facilitates the usage of mixed effect models by novice users who might not hold all of the statistical knowledge that is necessary in order to run such a models using existing software tools.
